### Hi there, I'm Santiago 👋

I am a Ph.D. candidate in Chemical Engineering, bridging materials physics and high-performance software engineering. Below are the key repositories I have built, collaborated or contributed to:

* 📊 **[ExtractData](https://github.com/salflorom1/ExtractData):** (Creator) Object-oriented Python pipeline to automate data extraction from massive simulation trajectories.
* 🌡️ **[EOS](https://github.com/GeordyJ/EOS):** (Creator) Python script for computing thermodynamic properties using complex equations of states.
* 🌡️ **[SetChainbuild](https://github.com/GeordyJ/SetChainbuild):** (Creator) Python script made to setup MC simulations for a molecular simulation program named Chainbuild.
* ⚛️ **[DensToolKit](https://github.com/jmsolano/denstoolkit):** (Contributor) C++ software suite of programs designed to analyze scalar and vector fields related to the electron density of a molecule.
* 💻 **[Parallelized MD Software](https://github.com/dianamcb/MD_MPI_Parallelized_Simulation_Soft_Matter_Systems):** (Collaborator) MPI Parallelized MD simulation program that predicts the thermodynamic properties of hard-spheres and Lennard-Jones fluids.
* 💻 **[Electrolyte MC Software](https://github.com/dianamcb/Monte-Carlo_ions-EDL):** (Collaborator) Monte Carlo simulation program in C++ aimed to study the electrical double layer dependence on ion size in electrolytes.
